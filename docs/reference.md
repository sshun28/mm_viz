# 9. 参考情報

### 9.1 マイクロマウス競技ルール
- **標準的な迷路サイズ**：16x16マスまたは32x32マスの正方形迷路が基本
- **壁の配置ルール**：
  - 外周は全て壁で囲まれている
  - 内壁は垂直または水平に配置
  - 一つのセルは最大4つの壁（北、東、南、西）で囲まれる
- **スタートとゴール位置**：
  - スタートは通常、左下（南西）の角
  - ゴールは大会毎に指定される
